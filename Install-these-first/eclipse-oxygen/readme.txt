Extract this ( eclipse-jee-2021-06-R-win32-x86_64 ) ZIP file,
And inside it, there is one folder named "eclipse"

Inside it, there is exectable file, after running, which will setup Older version of Eclipse( Eclipse Oxygen ) for Strut1 Framework

-> set Tomcat9 for server -> select jdk 8

-> after that click finish
